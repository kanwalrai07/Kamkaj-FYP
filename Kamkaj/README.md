## ğŸ Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Node.js](https://nodejs.org/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kanwalrai07/Kamkaj-FYP.git
   cd Kamkaj-FYP
   ```

2. **Setup the App:**
   ```bash
   cd kamkaj_app
   flutter pub get
   ```

3. **Setup the Backend (If running locally):**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

4. **Run the App:**
   ```bash
   cd kamkaj_app
   flutter run
   ```
